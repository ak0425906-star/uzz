# UZZ 🌕
### *Your memories, written in the stars.*

UZZ is a hyper-realistic, immersive digital sanctuary designed to immortalize human connection across celestial dimensions. It transforms linear time into a living, reactive constellation of memories, anchored by a photorealistic lunar core.

---

> [!IMPORTANT]
> **The Living Universe System**
> UZZ isn't just a journal — it’s an emotionally adaptive environment. UI elements breathe with life, stars react to your presence, and the cosmos shifts based on the "mood" of your shared history.

---

## 🌑 The Orbital Core

### 🌕 Photorealistic Lunar Anchor
The centerpiece of the experience. A high-fidelity, 3D-textured moon with craters, mare, and volumetric lighting that serves as the gravitational center for your digital world.

### ✨ StarField Particle Engine (V2)
A next-gen, GPU-accelerated canvas engine. 
- **Cursor Parallax**: Stars subtly shift as you move, creating a deep 3D immersion.
- **Multitone Tints**: Every star is unique, with tints of ice-blue, warm gold, and soft violet.
- **Atmospheric Nebulas**: Deep-space gradients that slowly drift and breathe in the background.

### 🧭 Constellation Navigation
Traditional menus are replaced by a mapping system. Active sections are highlighted with a glowing star marker, making navigation feel like charting a course through space.

## 🛠️ Technical Specifications

### 🏗️ The Architecture
- **Framework**: [Next.js 16.1.6](https://nextjs.org/) (Turbopack + App Router)
- **Engine**: [Framer Motion](https://www.framer.com/motion/) + Custom Canvas Rendering
- **Design**: Vanilla CSS Architecture + Tailwind (Utility Layer)
- **Auth**: NextAuth.js (Secure Celestial Credentials)
- **Data**: MongoDB Atlas (Vector-style memory storage)
- **PWA**: Fully installable with custom Lunar icons and push-notification protocols.

### ⚙️ Performance & UX
- **GPU Accelerated**: Smooth 60FPS animations and physics-based interactions.
- **Zero-Clutter Philosophy**: Extreme minimalism for low cognitive load and high emotional impact.
- **Glass-Card UI**: Floating panels with inner-glow and depth-based shadows.

## 🚀 Initialization Sequence

1. **Clone the coordinates**:
   ```bash
   git clone [repository-url]
   cd UZZ
   ```

2. **Fuel the systems**:
   ```bash
   npm install
   ```

3. **Configure Environment Variables**:
   Create a `.env.local` file:
   ```env
   # Storage & Database
   MONGODB_URI=your_mongodb_uri
   CLOUDINARY_CLOUD_NAME=your_cloudinary_name
   
   # Celestial Navigation
   NEXTAUTH_SECRET=your_nextauth_secret
   NEXTAUTH_URL=http://localhost:3000
   ```

4. **Launch into Orbit**:
   ```bash
   npm run dev
   ```
